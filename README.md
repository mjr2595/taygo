# TAYGO

[![Netlify Status](https://api.netlify.com/api/v1/badges/3e4bf7c1-4922-4a10-ae48-58e54df08d0b/deploy-status)](https://app.netlify.com/projects/taygo/deploys)

An interactive bingo-style game designed to be played while watching Torc streams. Taylor's Guidance Counselor podcast, Laid Off Lounge, Tech Commute, Friday Yappy Hour, and other random streams. Toggle between Taylor or Jason depending on who is streaming.
